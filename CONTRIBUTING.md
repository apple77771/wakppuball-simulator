# Contributing

버그 제보와 개선 제안을 환영합니다.

## 버그 제보

[Issues](https://github.com/apple77771/wakppuball-simulator/issues/new/choose)에서 양식을 선택하고 다음 내용을 포함해 주세요.

- 기기와 운영체제
- 브라우저 이름과 버전
- 문제를 재현하는 순서
- 예상한 결과와 실제 결과
- 가능하면 스크린샷 또는 짧은 영상

## 코드 기여

1. 저장소를 포크하고 작업 브랜치를 만듭니다.
2. `node server.js`로 로컬 실행을 확인합니다.
3. 데스크톱과 모바일 환경에서 변경 사항을 점검합니다.
4. 변경 이유와 검증 결과를 포함한 Pull Request를 생성합니다.

대규모 기능은 구현 전에 Issue에서 범위와 방향을 먼저 논의해 주세요.
